# Scooter Rental System

This is a web application for managing scooter rentals. The system allows users to rent different categories of scooters for varying durations, with prices depending on the rental period.

## Features

- User registration and authentication
- Admin dashboard for managing rentals
- Four scooter categories with different pricing structures
- Automated pricing calculations based on rental duration
- Rental history tracking
- Responsive design for desktop and mobile devices

## Scooter Categories

The system offers four categories of scooters:

1. **Category A**: Sym Orbit 125cc
   - 1 day: R400
   - 2-10 Days: R300/day
   - 11-29 days: R225/day
   - 30+ days: R120/day

2. **Category B**: Jet 14 200cc
   - 1 day: R450
   - 2-10 Days: R350/day
   - 11-29 days: R255/day
   - 30+ days: R150/day

3. **Category C**: Citycom 300cc
   - 1 day: R550
   - 2-10 Days: R500/day
   - 11-29 days: R350/day
   - 30+ days: R250/day

4. **Category D**: Vespa 150 or 300cc
   - 1 day: R850
   - 2-10 Days: R600/day
   - 11-29 days: R400/day
   - 30+ days: R250/day

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/mmhussein13/scooter_rental_system.git
   ```

2. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

3. Run the application:
   ```
   python app.py
   ```

4. Access the application at http://localhost:5000
